# MotoTrack

Motorrad-Service-Tracker: Tankbuch, Wartung, Reparaturen, Fahrzeugdaten und TÜV-Erinnerung — alles in einer einzigen HTML-Datei, Daten werden lokal im Browser gespeichert (LocalStorage).

## Nutzung

Einfach `index.html` im Browser öffnen, oder über GitHub Pages hosten:

1. Repo auf GitHub pushen
2. Unter Settings → Pages die Quelle auf den `main`-Branch (Root) stellen
3. Die App ist dann unter `https://<benutzername>.github.io/<repo-name>/` erreichbar

## Features

- Dashboard mit Service-Status ("FREE"-Regel), TÜV-Warnung und Kostenübersicht
- Tankbuch mit Verbrauchsberechnung
- Wartungs- und Reparatur-Protokoll
- Fahrzeugdaten (Modell, FIN, Reifendruck, Motoröl)
- Scheckheft-Export als PDF (Druckdialog)
- JSON-Backup (Export/Import) der lokal gespeicherten Daten
